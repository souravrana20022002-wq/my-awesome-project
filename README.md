# TaskFlow - Modern Task Management Application

TaskFlow is a polished React.js portfolio project that showcases a production-style task management experience with authentication, task CRUD, filtering, pagination, theming, and a responsive dashboard.

## Features

- Authentication UI with login and signup flows
- Protected routing and localStorage session persistence
- Dashboard with task statistics and profile summary
- Create, edit, delete, and complete tasks
- Task priorities, categories, and due dates
- Search, sort, filter, and pagination
- Responsive SaaS-style UI with dark/light theme toggle
- Error boundary and loading/empty states

## Tech Stack

- React.js + Vite
- JavaScript ES6+
- React Router DOM
- Context API
- Axios
- Tailwind CSS
- LocalStorage

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
